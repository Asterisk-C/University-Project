This is an abstract strategy board game which called Gobang. 

Players alternate turns placing a stone of their color on an empty intersection. The winner is the first player to form an unbroken chain of five stones horizontally, vertically, or diagonally.

There are 2 game modes: 1P and 2P.

1P is player vs computer, and 2P is player (pick yellow) vs player (pick black).

How to run the game: import the project to eclipse and run ChessUi.java.

All test class is in package test.
